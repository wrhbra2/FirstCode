APLICAÇÃO - REQUISITOS:
 * CADASTRO DE ALUNO 
	- ID
	- NOME COMPLETO
	- CÓDIGO DE MATRÍCULA -> ATRELADO AO PRONTUÁRIO DO ALUNO
	- REGISTRO DO ALUNO (RA) -> ATRELADO AO SED (SISTEMA DA SECRETARIA DA EDUCAÇÃO
 * CADASTRO DE FUNCIONÁRIOS:
	- ID
	- NOME COMPLETO
	- CARGO (DOCENTE; AGENTE DE ORGANIZAÇÃO ESCOLAR, ETC...)
	- TURMA (OBRIGATÓRIO CASO CARGO SEJA IGUAL A DOCENTE)
 * CADASTRO BIBLIOTECA
	- CÓDIGO DO LIVRO (ISBN)
	- TÍTULO
	- AUTOR
	- GÊNERO LITERÁRIO - COMBO LIST
	- SUBCATEGORIA - EX.: PORTUGUÊS
	- STATUS DE EMPRÉSTIMO (EMPRESTADO OU LIVRE)
	- DATA DE DEVOLUÇÃO (CASO STATUS EMPRESTADO, SENÃO, NULL)
	* CADASTRO DE GÊNERO LITERÁRIO:
		- ID 
		- DESCRIÇÃO
	* CONTROLE DE EMPRÉSTIMOS DE LIVROS
		- CÓDIGO DO LIVRO (ISBN
		- DATA DE EMPRÉSTIMO
		- DATA DE DEVOLUÇÃO - DEFAULT (+15 DIAS CORRIDOS)
		- ID DO REQUISITANTE (ALUNO OU DOCENTE)

* CADASTRO DE MATERIAIS ESCOLARES
	- ID DO MATERIAL
	- DESCRIÇÃO
	- ITENS QUE COMPÕE O MATERIAL (CABOS, CONECTORES E AFINS)
	- AVARIAS ATUAIS
	
	* CONTROLE DE AGENDAMENTO DE MATERIAIS ESCOLARES   **(PASSIVEIS DE ALTERAÇÕES OU CANCELAMENTOS (EXCLUSÕES) DE AGENDAMENTOS)
		- ID MATERIAL
		- ID DO FUNCIONÁRIO
		- LOCAL DE USO
		- DATA / HORA DE SAÍDA
		- DATA / HORA DEVOLUÇÃO

	* CONTROLE DE DEVOLUÇÕES DE MATERIAIS ESCOLARES
		- ID MATERIAL
		- FUNCIONÁRIO
		- ESTADO DO MATERIAL - DEVOLUÇÃO 
		- DATA/HORA DEVOLUÃO (CASO ATRASO)
